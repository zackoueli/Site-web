Dispo — Emploi & Recrutement

Politique de confidentialité

Dernière mise à jour : 19 juin 2026

BreizhApp s'engage à protéger votre vie privée. Cette politique explique quelles données sont collectées dans l'application Dispo, comment elles sont utilisées et quels sont vos droits en tant qu'utilisateur.

1. Éditeur de l'application

  • Éditeur : BreizhApp — Enzo Omnes, France
  • Application : Dispo — Emploi & Recrutement (fr.dispo.app)
  • Site web : breizhapp.tech
  • Contact : breizhapp@outlook.fr

2. Données collectées

Dispo collecte deux catégories de données selon le type de compte créé.

Compte candidat

  • Identité : prénom, nom, âge
  • Localisation : ville, code postal
  • Coordonnées : adresse email, mot de passe (chiffré), numéro de téléphone (optionnel)
  • Profil : photo de profil, domaine professionnel, familles de métiers, formation, expériences
  • Recherche d'emploi : type de contrat recherché, rythme de travail, permis de conduire, description libre, statut de disponibilité

Compte entreprise

  • Nom de l'entreprise
  • Coordonnées : adresse email, mot de passe (chiffré), numéro de téléphone (optionnel)
  • Localisation : ville

Données communes

  • Messages échangés entre candidats et entreprises
  • Token de notification push (pour la réception des messages)

3. Utilisation des données

Vos données sont utilisées exclusivement pour :

  • Créer et gérer votre compte utilisateur
  • Mettre en relation les candidats et les entreprises recruteuses
  • Afficher les profils candidats aux entreprises (et inversement)
  • Permettre la messagerie entre candidats et entreprises
  • Envoyer des notifications push lors de la réception de nouveaux messages
  • Assurer la sécurité et le bon fonctionnement de l'application

4. Partage des données

BreizhApp ne vend aucune donnée personnelle à des tiers. Certaines données sont partagées uniquement dans le cadre du fonctionnement de l'application :

Visibilité entre utilisateurs

  • Les profils candidats (y compris le numéro de téléphone s'il est renseigné) sont visibles par les entreprises inscrites sur la plateforme
  • Les profils entreprises sont visibles par les candidats

Services tiers

  • Firebase (Google LLC) — authentification, base de données (Firestore), stockage des photos (Storage), notifications push (Cloud Messaging)
  • Expo (Expo Inc.) — gestion des notifications push sur iOS et Android

Ces prestataires traitent vos données uniquement pour fournir leurs services techniques et sont soumis à leurs propres politiques de confidentialité.

5. Stockage et sécurité

  • Les données sont stockées sur les serveurs Firebase (Google) avec chiffrement en transit (TLS) et au repos
  • Les photos de profil sont hébergées sur Firebase Storage
  • Les mots de passe sont chiffrés via Firebase Authentication et ne sont jamais stockés en clair
  • Des règles de sécurité Firestore garantissent que chaque utilisateur n'accède qu'aux données auxquelles il est autorisé

6. Conservation des données

Vos données sont conservées tant que votre compte est actif. En cas de suppression de compte, toutes vos données personnelles sont effacées définitivement. Pour demander la suppression de votre compte et de vos données, contactez-nous à breizhapp@outlook.fr.

7. Permissions demandées

  • Accès à la galerie photos — pour choisir une photo de profil
  • Accès à la caméra — pour prendre une photo de profil
  • Notifications push — pour recevoir les alertes de nouveaux messages

Ces permissions sont demandées uniquement au moment où vous utilisez les fonctionnalités correspondantes et peuvent être révoquées à tout moment dans les paramètres de votre appareil.

8. Vos droits (RGPD)

Conformément au Règlement Général sur la Protection des Données (RGPD), vous disposez des droits suivants :

  • Accès : obtenir une copie des données vous concernant
  • Rectification : corriger des données inexactes ou incomplètes
  • Suppression : demander l'effacement de votre compte et de vos données
  • Opposition : vous opposer au traitement de vos données
  • Portabilité : recevoir vos données dans un format structuré et lisible

Pour exercer ces droits : breizhapp@outlook.fr

Vous disposez également du droit d'introduire une réclamation auprès de la CNIL (Commission Nationale de l'Informatique et des Libertés).

9. Mineurs

Dispo est une application de mise en relation professionnelle destinée aux personnes en âge de travailler. Elle n'est pas destinée aux enfants de moins de 16 ans. Nous ne collectons pas sciemment de données personnelles concernant des mineurs de moins de 16 ans.

10. Modifications

Toute modification importante de cette politique sera notifiée dans l'application. La date de dernière mise à jour est indiquée en haut de cette page.
